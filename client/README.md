# Project Titan Client
This is the front-end client for Project Titan. This project builds the html, js, and css files that will be deployed to a hosting site such as AWS S3.

## Running Project
Install depedencies:
```npm install```

Start server locally:
```npm run start```

Run unit tests:
```npm run test```

## Project Layout
  - Components
  - Containers
  - Store
    - Actions
    - Selectors
    - Sagas
    - Reducers

## App Views
- Ranking Board
- Match Recording
- Match Creation Form
